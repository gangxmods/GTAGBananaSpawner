# GTAGBananaSpawner
features
This is a simple mod that allows you to spawn bananas
from your hand. To spawn the bananas hold left trigger
and it will spawn the bananas based on your settings on an ingame board.
                                                 

Requirement
MOST RECENT UTILLA
